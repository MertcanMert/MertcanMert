## 🧠 About me

I design **production-grade backend systems** for SaaS — with a focus on:

- 🧩 **API contracts & response consistency** — deterministic, predictable APIs
- 🛡️ **Error management & observability** — centralized, ELK-friendly logging
- 🏗️ **Scalable NestJS architecture** — modular, multi-tenant, security-first
- 🤖 **Developer tooling & AI integrations** — Claude Code & Cursor automation

> I don't build demos. I build **foundations that survive production**.

🏢 Founder of **[Armert Labs](https://github.com/Armert-Labs)** · 🌐 **[armert.com.tr](https://armert.com.tr)** · 📍 Türkiye
